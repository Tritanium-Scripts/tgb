1. Einführung
Vielen Dank, dass Sie sich für dieses Gästebuch entschieden haben. Ich hoffe, es entspricht Ihren Vorstellungen!
Bitte beachten Sie, dass dies noch eine Betaversion ist, das heißt, manche Features funktionieren noch nicht oder noch nicht richtig/vollständig und auch sonst kann es sein, dass es noch den einen oder anderen Fehler gibt. Weitere Informationen dazu erhalten sie unter Betahinweise.



2. Installation

2.1 Vorrausetzungen
Dieses Gästebuch stellt keine hohen Anforderungen an ihren Server. Lediglich benötigt werden:
- PHP 4 (Das Gästebuch wurde unter PHP 4.2.0 entwickelt, es sollte aber auch unter früheren Versionen laufen)
- ca. 0,5-1 MB Webspace (aber auch mehr bzw. weniger, je nach Anzahl/Länge der Einträge)

2.2 Voreinstellungen
Bevor sie mit dem Upload beginnen, sollten Sie die Datei "settings.php" mit einem beliebigen Texteditor öffnen und die einzelnen Punkte auf ihre Angaben abändern. Was die einzelnen Dinge bewirken, steht jeweils in der Zeile darüber. Sind Sie damit fertig, können Sie mit dem Upload beginnen. Es ist übrigens nicht unbedingt notwendig, jetzt schon alle Einstellungen vorzunehmen. Sie können dies später jederzeit nachholen, müssen dann aber die Datei "settings.php" neu hochladen und die alte überschreiben.

2.3 Upload
Nachdem Sie alle Einstellungen vorgenommen haben, beginnen Sie mit dem Upload per FTP auf ihren Server. Beachten Sie, dass es zwei Übertragungsmodi (Binary und Ascii) gibt. Normalerweise sollte ihr FTP-Programm jeweils selber den richtigen Modus wählen, aber falls Zweifel bestehen, können Sie diesen jeweils auch manuell wählen (wie das bei ihrem FTP-Programm genau geht, sollte in dessen Anleitung stehen):
Verzeichnis "data" (+ Inhalt) Binarymodus
Verzeichnis "images" (+ Inhalt) Binarymodus
Verzeichnis "styles" (+ Inhalt) Asciimodus
Dateien *.php Asciimodus

2.4 Rechte setzen
Sind alle Verzeichnisse und Dateien hochgeladen, müssen Sie noch spezielle Rechte verteilen. Diese Option sollten Sie in ihrem FTP-Programm unter dem Punkt "CHMOD" finden. Erteilen Sie folgenden Dateien/Ordnern folgende Rechte:
Verzeichnis "data" 777
Dateien *.dat (in "data") 777
Dateien *.php 755

2.5 Erste Schritte
Ihr Gästebuch sollte nun korrekt installiert sein. Sie können es nun in Ihrem Webbrowser per index.php aufrufen (z.B. http://www.meinedomain.de/gb/index.php). Die Administration dagegen erreichen Sie über admin.php (z.B. http://www.meinedomain.de/gb/admin.php). Öffnen Sie auch genau diese und loggen Sie sich mit dem Zugangscode "ReiAyanami" (ohne Anführungszeichen) ein. Beachten sie Groß- und Kleinschreibung! Nun sollten Sie als erstes links in der Navigation der Administration den Punkt "Zugangscode ändern" auswählen. Dort geben Sie als aktuellen Zugangscode wieder "ReiAyanami" ein. Als neuen Zugangscode sollten Sie möglichst eine wirre Kombination aus Zahlen, Buchstaben und anderen Zeichen wählen, damit es unbefugten Personen möglichst schwer (vielleicht sogar fast unmöglich) gemacht wird, auf diesem Wege das Gästebuch zu manipulieren. Haben Sie dies getan, steht Ihnen nun ein sehr komfortables und umfangreiches Gästebuch für Ihre Homepage zur Verfügung! :)



3. Betahinweise
Diese Version des Gästebuchs ist eine Betaversion, d.h. es befindet sich höchstwahrscheinlich noch der eine oder andere Fehler im Script und einige Features funktionieren auch noch nicht oder noch nicht vollständig. Im Speziellen gehören hier folgende Dinge dazu: ICQ, AIM, Y! und die Emailbenachrichtigungen. Für die ersten 3 Punkte gilt: Die Daten kann man zwar beim Eintragen angeben und sie werden auch gespeichert, allerdings werden sie später im Gästebuch noch nicht angezeigt. Für die Emailbenachrichtigungen gilt: Diese Funktionen sind komplett integriert, allerdings noch nicht getestet. Ich kann also nicht garantieren, dass die Emails vom Script her auch ankommen (theoretisch sollte es schon so sein, aber man weiß ja nie...). Falls Ihnen hierbei etwas auffällt, können Sie mir das oder auch andere Fehler, Verbesserungsvorschläge usw. gerne per Mail mitteilen oder es einfach in das Forum auf meiner Homepage schreiben.
